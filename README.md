# Notes CRUD App (React + Django DRF)

## 🚀 Features
- Create Notes
- View Notes
- Update Notes
- Delete Notes

## 🛠 Tech Stack
- React
- Django
- Django REST Framework

## 📡 API Endpoints
- GET /api/notes/
- POST /api/notes/
- PUT /api/notes/<id>/
- DELETE /api/notes/<id>/

## 💻 How to Run

### Backend
python manage.py runserver

### Frontend
npm run dev
